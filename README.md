## 프로젝트 소개

'Tech-Mail'은 매주 정해진 시간에 개발 지식을 제공하는 이메일 기반 서비스입니다.

[✉️ 홈페이지 ](https://www.tech-mail.site)

## 기술 스택

- Java 21
- Gradle 
- Spring Boot 3.4.0
- Spring Data JPA
- Caffeine Cache
- SMTP
- QueryDSL 
- MySQL 
- AWS(EC2, RDS)
- Nginx
- Docker, docker-compose
- GitHub Actions
